# heart-disease-prediction
# ❤️ Heart Disease Prediction using Logistic Regression


## 📊 Project Overview

This project implements a **Logistic Regression** model to predict heart disease using the UCI Heart Disease dataset. It includes comprehensive data preprocessing, exploratory data analysis (EDA), feature importance analysis, and clinical insights visualization.

### Key Achievements
- **86.9% Accuracy** on test data
-  **ROC-AUC Score: 0.92** (Excellent discrimination)
-  **90% Recall** for heart disease detection
-  **Balanced performance** across both classes

## 📈 Model Performance

| Metric | Score |
|--------|-------|
| **Accuracy** | 86.9% |
| **ROC-AUC** | 0.92 |
| **Precision (Disease)** | 0.84 |
| **Recall (Disease)** | 0.90 |
| **F1-Score (Disease)** | 0.87 |
